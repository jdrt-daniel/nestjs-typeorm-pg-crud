# Description

A small example with postgres and swagger for CRUD functions

# Run application

- Execute the command ```npm install```
- Copy de ```.env-example``` and rename form ```.env.[...]```

  ```bash
    .env.development
    .env.production
  ```

- Execute the command ```docker compose up -d```
- Run project with ```npm run dev```
  
  ```
    npm run dev
    npm run prod
  ```
